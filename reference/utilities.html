# Utilities

HiQ provides numerous custom selectors, mixins and utility classes to make styling your project easier and faster. See below for a complete reference.

# Custom Selectors

Behind-the-scenes, HiQ uses custom selectors, made possible by the postcss-custom-selectors plugin. If you use the HiQ source files with PostCSS, these are also available to you.

# Mixins

HiQ mixins are available for you to use if you use the source files with PostCSS. Most mixins are also available as classes. Keep in mind that you should typically only use HiQ utility classes if that utility will apply at all breakpoints.

@mixin/.classDescription
container
.container		Makes an element a HiQ container.
is-fluid
.is-fluid		Makes a container the full width of its parent.
flex-grid
Makes an element a wrapping flex-grid. Accepts $flex-basis and $grid-basis as arguments.
is-full-width
.is-full-width		Makes an element the full width of its parent.
is-max-full-width
.is-max-full-width		Makes an element no more than the width of its parent.
is-centered
.is-centered		Centers an element within its parent.
is-pulled-left
.is-pulled-left		Floats an element left.
is-pulled-right
.is-pulled-right		Floats an element right.
is-clearfix
.is-clearfix		Applies the clearfix hack to an element.
absolute-stretch
.absolute-stretch		Stretches an element to fill its parent.
overflow-touch
.overflow-touch		Applies momentum-based scrolling for overflowing elements on touch devices.
is-resizable
Makes an element resizable.
is-visually-hidden
.is-visually-hidden		Visually hides an element but keeps it accessible to screen readers.
is-size-1
.is-size-1		Makes an element's text the first size in the HiQ type scale.
is-size-2
.is-size-2		Makes an element's text the second size in the HiQ type scale.
is-size-3

.is-size-4		Makes an element's text the fourth size in the HiQ type scale.
is-size-5

is-size-6
.is-size-6		Makes an element's text the sixth size in the HiQ type scale.
is-large
.is-large		Makes an element's text the 'large' size in the HiQ type scale.
is-small
.is-small		Makes an element's text the 'small' size in the HiQ type scale.
has-font-weight-light
.has-font-weight-light		Gives an element's text light weight.
has-font-weight-normal
.has-font-weight-normal		Gives an element's text normal weight.
has-font-weight-medium
.has-font-weight-medium		Gives an element's text medium weight.
has-font-weight-semibold
.has-font-weight-semibold		Gives an element's text semibold weight.
has-font-weight-bold
.has-font-weight-bold		Gives an element's text bold weight.
is-unstyled
.is-unstyled		Removes the default browser styling from a list element.
has-inline-children
.has-inline-children		Makes an list's items display in a horizontal line.
has-text-left
.has-text-left		Aligns an element's text to the left.
has-text-centered
.has-text-centered		Centers an element's text.
has-text-right
.has-text-right		Aligns an element's text to the right.
has-text-justified
.has-text-justified		Justifies an element's text.
is-capitalized

is-lowercase

is-uppercase

has-text-truncated

is-clipped
.is-clipped		Hides overflow that overflows element bounds.
is-marginless
.is-marginless		Removes any margin from an element.
is-paddingless

is-radiusless

is-shadowless
.is-shadowless		Removes any box shadow from an element.
is-initial
Resets all properties on an element to their initial values.
is-unset
Resets all properties on an element to inheritable values, or else initial values.
button
Styles an element like a button.
input
Applies HiQ's input styles to an element.
is-controlled
Removes default browser styling from a button or input.
is-unselectable
.is-unselectable		Prevents a user from selecting an element or its text.
