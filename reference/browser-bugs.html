# Browser Bugs

Reference the complete list of browser bugs related to the HiQ framework. Note that most of these are general bugs not directly tied to HiQ styles.

BugBrowsers
Fluid typography calculated using calc() and viewport units is currently not supported in Safari and Internet Explorer. Min and max font-size will still be respected.Safari, IE
The details element is currently not supported in Edge or Internet Explorer.Edge, IE
Inputs with display: inline-flex applied currently do not display correctly in a line in Edge or Internet Explorer.Edge, IE
There are currently some issues styling the draggable button on range inputs in Edge, specifically related to sizing and border-radius.Edge
Applying button styling to file inputs is currently not supported in Edge or Internet Explorer.Edge, IE
The meter element is currently not supported in Internet Explorer.Edge, IE
Indeterminate progress bar styling is currently not supported in Edge or Internet Explorer.Edge, IE
